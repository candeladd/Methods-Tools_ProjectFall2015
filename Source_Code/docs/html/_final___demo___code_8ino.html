Detailed Description

This is the final demo code.

We used this in the presentation.

This code sweeps the base of the robot, then the upper arm, then the claw. The code loops infinitely until power is cut from the arduino or the shield.

Function Documentation

void loop ( )

This code loops through every cycle.

keeps cycling until power is disconnected.

Calls the sweep function twice for each joint. There are delays in between each loop to ensure that the servo doesn't move too fast and burn out. There are different servo min and maxes based on the physical arcitecture of the arm. We ensured that each part moved within range of the table, etc.
