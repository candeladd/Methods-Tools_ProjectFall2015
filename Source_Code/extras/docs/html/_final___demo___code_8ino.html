Detailed Description

This is the final demo code.

We used this in the presentation.

This code sweeps the base of the robot, then the upper arm, then the claw. The code loops infinitely until power is cut from the arduino or the shield.

Function Documentation

void loop ( )

This code loops through every cycle.

keeps cycling until power is disconnected.

There are two sets of for loops for each joint. The first for loop sweeps the servo up, the second for loop sweeps the servo down. There are delays in between each loop to ensure that the servo doesn't move too fast and burn out. There are different servo min and maxes based on the physical arcitecture of the arm. We ensured that each part moved within range of the table, etc.

void setup ( )

This sets up the Arduino code.

Runs only once at the beginning.

Serial.begin(9600) sets up the serial monitor for debugging purposes. The next line simply prints to the serial monitor. The last chunk sets up the servo shield. It sets up the pwm and wires in the shield.
